Legos and Software:
putting all the pieces together

Posted 28 May 2018

So what do legos and software have in common? Quite a bit in my humble opinion. The world of building software offers one such solution that lets you add the dependencies you need, and keep out the ones you don't. With over 700,000 free open-source packages, npm has created a community around full-stack JavaScript applications.

Package manager npm for Node.js makes it easy to build custom, flexible applications. Build managers like Gulp and development tools like Browsersync provide a powerful development environment.

By organizing gulp tasks in a tiered manner another build is one hotkey away with little or no wait in the browser. Or in the case of CSS, a new file is injected directly into the browser without a view change or page reload. Watching elements just move insights joy in my heart no matter how many times I’ve seen it. This coffee mug is a wonderful bit of context.

Front side frameworks like Angular and React further allow modularization buy breaking down components and their associated logic into bite size pieces. Allowing teams to more easily divide up what needs to get done. As the pieces begin working together something truly special can be born.
