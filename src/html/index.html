Angular / Ruby on Rails

Toolie Box

As developers increase their skill set it's all too often easy to start to have a hard time remembering all of the tools they have in their “toolbox”. Enter Toolie Box, a cloud based repository of tools, tips, and tricks for common programing languages in use today. Complete with your own 'Workbench' and live coding test windows powered by Relp.it, this application keeps track of tools that make dev’s lives easier! A two server system delivers content to the client: 1) Node/Express serving static assets and Angular front-end, 2) Ruby on Rails API with a Postgresql database for user data storage.

TimeSheetsIO

Developed to fill specialized needs for property management companies, TimeSheetsIO revolutionizes the way hospitality employees keep track of their daily time by replacing a paper-based system. Instead of piles of paper, users now have an intuitive interface to enter daily time activities. Administrators can then generate reports with filters to drill into the information they need for billing. This project is a proof of concept for a developer building their skill set and does not incorporate a frontend framework that would be required for a production ready app
